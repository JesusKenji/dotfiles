# TMUX plugins

#### Install dependencies
- Debian: `sudo apt install xsel #or xclip`
- MacOSX: `brew install reattach-to-user-namespace`

# TMUX cheat sheet

## Sessions
### In opened session press
`ctrl+j d  (detach)`

### Attach session
`$ tmux attach -t session_name`

### Switch between sessions
`ctrl+j s`
